TO COMPILE "DESERT WANDER - THE GAME" ALL YOU NEED TO DO IS:

1. WHILE IN THE "CPSC4050-FINAL-PROJECT" FOLDER, OPEN THE "MAIN" FOLDER
2. IN THE TERMINAL RUN "make"
3. THE MAKEFILE PRODUCED AN EXECUTABLE CALLED "DesertWanderer"
4. TO RUN THE APPLICATION, RUN "./DesertWanderer" WHILE STILL IN THE MAIN FOLDER DIRECTORY IN THE TERMIAL
