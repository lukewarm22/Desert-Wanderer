TO COMPILE "DESERT WANDER - THE GAME" ALL YOU NEED TO DO IS:

1. OPEN "CPSC4050-FINAL-PROJECT"
2. OPEN THE "MAIN" FOLDER
3. IN THE TERMINAL RUN "make"
4. THE MAKEFILE PRODUCED AN EXECUTABLE CALLED "DesertWanderer"
5. TO RUN THE APPLICATION, RUN "./DesertWanderer" WHILE STILL IN THE MAIN FOLDER DIRECTORY IN THE TERMIAL
